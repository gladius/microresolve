## What is MicroResolve?

MicroResolve maps natural language queries to your defined intents using a fast lexical pipeline — no neural network, no embeddings, no cold inference latency.

```
"I want to cancel my order"  →  cancel_order  (score: 0.91)  ~30µs
"where is my package?"       →  track_order   (score: 0.87)  ~28µs
```

It learns from corrections. The more queries it sees, the better it gets — without any retraining step.

## Benchmark results

| Dataset | Cold start | After learning | Latency |
|---------|-----------|----------------|---------|
| CLINC150 (150 intents) | 74.8% exact | **90.3% exact** | ~2.8ms HTTP |
| SaaS support (10 intents) | 58% exact / 70.7% F1 | **71% exact / 88.6% F1** | ~640µs |
| MCP tool routing (100 tools) | 44.7% exact / 89.4% recall | **48.9% exact / 94.7% recall** | ~870µs |
| Multilingual (5 languages) | 68% exact / 86% recall | cold only | ~950µs |
