# Rust Examples

## Setup

```bash
# From the project root
cargo run --example rust_basic
```

No additional dependencies needed — examples use the library directly.

## Examples

| Example | Description |
|---------|-------------|
| `rust_basic.rs` | Routing, multi-intent, learning, export/import, discovery |

Note: Rust examples live in the parent `examples/` directory (Cargo convention requires `examples/*.rs`).
